= Docker Compose


== up - start
----
docker-compose up -d
----

-d 백그라운드 실행

== down - shutdown
----
docker-compose down
----

== ps

----
docker-compose ps
----

== logs
----
docker-compose logs
----

== docker-compose run

----
docker-compose run kafka /bin/bash
----
