== 외부에서 설정 주입

=== application.properties
----
server.port=8080
----
커맨드 라인에서 --server.port=9000 하면 9000포트를 사용하게 된다.


----
server.port=${port:8080}
----
이렇게 하면 커맨드 라인에서 --port=9000 하면 9000포트를 사용하게 된다.
