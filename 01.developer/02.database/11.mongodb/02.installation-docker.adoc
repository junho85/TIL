= mongodb installation - docker


[source]
----
docker run --name some-mongo -d mongo:tag
----

127.0.0.1: 을 넣어 주면 서버 내부에서만 접근 가능합니다.

[source]
----
$ docker run --name mymongo -p 127.0.0.1:27017:27017 -d mongo
----

[source]
----
$ docker start mymongo
----

[source]
----
$ mongo
$ mongo --port 27017
$ mongo localhost
----
