== 창 관련

=== Cmd + Shift + F12 Maximize Editor
Editor 영역을 최대크기로 펼침. 다시 누르면 원래대로 돌아옴.
파일 트리나 Gradle, Run 창 등을 숨기고 싶을 때 유용하다.

Shift + ESC Hide Active Tool Window 기능을 이용해서 숨길 수도 있다.


=== Zoom - Ctrl + Cmd + =
* 창 사이즈를 화면 크기에 맞춰 키운다.다시 누르면 기존 사이즈로 돌아 온다.
