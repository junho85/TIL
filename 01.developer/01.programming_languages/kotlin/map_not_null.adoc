= mapNotNull

map, mapNotNull 은 컬렉션의 각 요소에 대한 변환을 수행하는 함수

map 은 주어진 변환 함수를 사용하여 컬렉션의 각 요소를 변환하고, 변환된 결과를 새로운 컬렉션으로 반환

mapNotNull 함수는 변환 함수를 사용하여 각 요소를 변환하고, 변환 결과가 `null`이 아닌 경우에만 새로운 컬렉션에 추가
