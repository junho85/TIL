# perl IDE

## IntelliJ
* Perl Plugin 을 설치 한다.
  * https://plugins.jetbrains.com/plugin/7796-perl
  
* 설치해 두면 좋을 플러그인들
  * Template Toolkit
    * Template Toolkit 을 사용한다면 Template Toolkit 플러그인도 설치 한다. 확장자는 tt 로 한다.
    * https://plugins.jetbrains.com/plugin/11335-template-toolkit-support
    * https://github.com/Camelcade/Perl5-IDEA/wiki/Template-Toolkit-2
  * SQL
    * SQL 플러그인도 설치 해 두는게 좋음
    * Perl -> Injection Marker 에 가보면 지원 언어들 리스트 나오는데 SQL 은 지원하는게 좋음
