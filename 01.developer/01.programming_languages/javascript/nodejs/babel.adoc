= babel

https://babeljs.io/


== installation

----
npm install -g babel
----

babel-cli 설치
----
npm install --save-dev -g babel-cli
----

== 프로젝트에 넣기
----
npm install --save-dev babel-core
npm install --save-dev @babel-core
----
