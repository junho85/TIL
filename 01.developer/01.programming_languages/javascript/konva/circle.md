# Circle
* x, y 좌표와 radius (반지름) 값으로 원을 만들 수 있다.

```javascript
var circle = new Konva.Circle({
  x: stage.width() / 2,
  y: stage.height() / 2,
  radius: 70,
  fill: 'red',
  stroke: 'black',
  strokeWidth: 4
});
```
