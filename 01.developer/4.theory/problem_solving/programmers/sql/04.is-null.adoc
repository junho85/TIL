== 이름이 있는 동물의 아이디

https://programmers.co.kr/learn/courses/30/lessons/59407

----
SELECT animal_id
FROM ANIMAL_INS
WHERE name is not null
order by animal_id
----

== NULL 처리하기

https://programmers.co.kr/learn/courses/30/lessons/59410


동물의 생물 종, 이름, 성별 및 중성화 여부를 아이디 순으로 조회
이름이 없는 동물은 "No name"으로 표시

ifnull함수를 이용합니다.

----
select animal_type,
       ifnull(name, 'No name'),
       sex_upon_intake
from animal_ins
order by animal_id
----
