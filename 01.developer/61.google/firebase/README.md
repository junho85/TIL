# Firebase

프로젝트 추가


Hosting
```
$ npm install -g firebase-tools
```

Google 로그인
```
$ firebase login
```

프로젝트 시작
```
$ firebase init
```

웹사이트 배포
```
$ firebase deploy
```

```
$ firebase init hosting
```
