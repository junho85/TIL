## Primitive type Vs Reference Type
* 기본 타입과 참조 타입
* Primitive type
  * stack 에 저장
  * 논리형: boolean
  * 정수형: byte, short, int, long
  * 실수형: float, double
  * 문자형: char
* Reference type
  * heap 에 저장
  * 배열, 열거형 클래스, 인터페이스
  * Integer, Double...
