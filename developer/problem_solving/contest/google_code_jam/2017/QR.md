## Oversized Pancake Flipper
* [Problem A. Oversized Pancake Flipper](https://code.google.com/codejam/contest/3264486/dashboard)


```
Input
3
---+-++- 3
+++++ 4
-+-+- 4
```
T 3 은 테스트 케이스가 3개
-와 +로 구성된 문자열과 숫자 K

초코칩이 있는 happy side (+)
아무것도 없는 blank side (-)

K 는 한번에 뒤집을 수 있는 갯수




```
Output
Case #1: 3
Case #2: 0
Case #3: IMPOSSIBLE
```
