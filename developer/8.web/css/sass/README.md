# SASS
* CSS 와 비슷한데 좀 더 다양한 기능들을 한다.
* nest 구조의 CSS 를 좀 더 간단하고 보기 좋게 만들 수 있다.
* module 을 만들어서 불러 와서 사용 할 수 있다.
* 연산자를 이용한 연산이 가능하다.

## installation
```
brew install sass/sass/sass
```

## 연산자를 이용한 연산
SCSS
```
width: 300px / 960px * 100%;
```

CSS
```
width: 31.25%;
```
