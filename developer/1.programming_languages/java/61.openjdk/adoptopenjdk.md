# adoptopenjdk
* https://adoptopenjdk.net/index.html

## installation
```
brew tap AdoptOpenJDK/openjdk
brew cask install adoptopenjdk8
```

설치 위치: /Library/Java/JavaVirtualMachines/adoptopenjdk-8.jdk/Contents/Home
