## 8.1 고차 함수 정의
고차 함수는 다른 함수를 인자로 받거나 함수를 반환하는 함수

예를 들어 표준 라이브러리 함수인 filter는 술어 함수를 인자로 받으므로 고차 함수다.
```kotlin
list.filter { x > 0 }
```
