# yarn
* https://yarnpkg.com/lang/en/

## installation
* https://yarnpkg.com/en/docs/install

brew 로 설치 하기
```
brew install yarn
```

npm 으로 설치하기
```
npm install --global yarn
```

## update
```
yarn self-update
```
