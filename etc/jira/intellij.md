# jira - intellij 와 연동
## 기본 사용법
* Preferences -> Tools -> Tasks -> Servers
* '+' -> JIRA
* Server URL, Username, Password 입력 후 Test
* Search 는 기본값 써도 되고 원하면 바꿔 써도 됨
* JIRA 티켓 생성은 사이트에서 직접 해야 됨
* Open Task - opt + shift + n
* Close Active Task - opt + shift + w

## 기타
* Switch Task - opt + shift + t
