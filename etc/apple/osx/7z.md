# 7zip
installation
```
brew install p7zip
```

compress
```
7z a 20190927.zip 20190927.mp4
```
extract
```
7z x 20190927.zip 20190927.mp4
```

분할
```
7z a 20190927.zip 20190927.mp4 -v3500m
```
