# gnu utils
* coreutils - echo, cp 등
* findutils - find 등 
```
brew install coreutils findutils
```

/usr/bin/find

osx find 는 뒤에 꼭 . 을 붙여 줘야 현재 경로 기준으로 검색진행.

findutils 의 find 는 . 안붙여 줘도 됨.
