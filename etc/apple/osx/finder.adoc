= finder

== 강제 종료
* Cmd + Opt + Shift + ESC로 강제 종료
* Activity Monitor에서 강제 종료
* 터미널: killall Finder
* Force Quit Applications
* GUI에서 종료: 사과 -> Shift누르면 Force Quit Finder 나타남.
* 독의 실행 아이콘 Opt + 마우스 우클릭 후 Relaunch
