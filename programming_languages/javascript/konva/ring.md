# Ring
* 고리(ring)모양을 만들려면 Konva.Ring() 으로 만든다.
```javascript
var ring = new Konva.Ring({
  x: stage.width() / 2,
  y: stage.height() / 2,
  innerRadius: 40,
  outerRadius: 70,
  fill: 'yellow',
  stroke: 'black',
  strokeWidth: 4
});
```
