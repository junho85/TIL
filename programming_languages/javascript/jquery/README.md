# jQuery

## $(document).ready()
* [$(document).ready()](https://learn.jquery.com/using-jquery-core/document-ready/)
```javascript
// A $( document ).ready() block.
$( document ).ready(function() {
    console.log( "ready!" );
});
```

```javascript
// Shorthand for $( document ).ready()
$(function() {
    console.log( "ready!" );
});
```
