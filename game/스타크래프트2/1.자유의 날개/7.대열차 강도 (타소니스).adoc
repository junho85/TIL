= 7. 대열차 강도

임무 목표: 자치령 열차 가로채기
자금 보상: 110,000
연구 기회: +3 저그 연구

== 아주 어려움 공략

=== 코브라 열심히 뽑기
코브라 열심히 뽑고 해병 적당히 섞어 준다. 공중 유닛이 종종 나오기 때문에 공중 공격할 유닛이 필요하다.

=== 공성전차 이용하기
정문 돌파 (질)을 먼저 했다면 공성전차를 이용할 수 있다.
공성전차를 중간중간 잘 배치해 두면 호위 병력이나 불곰 부대도 별로 어렵지 않다.
